Bulk unfavorite all the albums in your Spotify library.

## Requirements

Setup at [developer dashboard](https://developer.spotify.com/dashboard).
Add callback: `http://127.0.0.1:4001/callback`.

## Usage

- load `.env` with direnv or desired tool.
- `go run main.go`
